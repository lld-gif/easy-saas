## Pre-Launch (4-6 weeks before)
- [ ] Landing page with waitlist + early access form
- [ ] Build in public: weekly updates on X, LinkedIn, Indie Hackers
- [ ] Create documentation site (use Mintlify or Docusaurus)
- [ ] Record a 2-minute demo video
- [ ] Collect 50+ email subscribers before launch
- [ ] Secure 5-10 beta customers for feedback

## Launch Campaign
- [ ] Product Hunt launch with a hunter (find one on PH's community)
- [ ] Hacker News "Show HN" post
- [ ] Reddit posts across 5+ subreddits
- [ ] X/Twitter launch thread (10+ tweets)
- [ ] LinkedIn article about the problem you're solving
- [ ] Email blast to waitlist
- [ ] Reach out to 5 niche newsletters for features
- [ ] Submit to startup directories (BetaList, SaaSHub, AlternativeTo)

## Post-Launch (Month 1-3)
- [ ] Weekly feature releases based on user feedback
- [ ] Start content marketing: 1 blog post per week
- [ ] Set up SEO basics: sitemap, meta tags, blog
- [ ] Build integrations with tools your users already use
- [ ] Implement referral program or invite system
- [ ] Track MRR, churn, and customer acquisition cost
